## 📌 Project Description
This is a To-Do List application built using JavaScript. It allows users to add, complete, and delete tasks. The application uses localStorage to persist data, so tasks remain even after refreshing the page.

---

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- localStorage

---

## ✨ Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Tasks persist after page refresh
- Clean modular JavaScript structure
